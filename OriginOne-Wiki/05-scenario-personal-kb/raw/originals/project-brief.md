# 项目 Brief：OriginOne 任务管理助手

## 目标

做一个给个人开发者使用的全栈项目管理助手。它能把会议记录、需求草稿、接口说明、规则文档放进 LLM-Wiki，再自动生成需求、任务拆分和验收清单。

## 核心用户

- 独立开发者。
- 小团队技术负责人。
- 需要同时维护产品、后端、前端和文档的人。

## MVP 功能

1. 创建项目。
2. 导入原始材料。
3. 为每份材料生成 source card。
4. 编织 project wiki、规则页、接口页。
5. 根据 wiki 生成需求草案和任务拆分。

## 非目标

- 暂不做多人权限。
- 暂不做复杂甘特图。
- 暂不接入真实生产数据库。
