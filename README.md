# OriginOne Demo

This repository contains runnable OriginOneAI demo projects.

## Demos

- `OriginOne-Wiki/`: a terminal-first LLM-Wiki 0-1 demo that starts from `raw/`, `wiki/`, and `output/`, then evolves into data-development and project-knowledge-base scenarios.

## Current Checks

The `OriginOne-Wiki` demo includes:

- runnable Python scripts under `OriginOne-Wiki/scripts/`
- terminal screenshots under `OriginOne-Wiki/assets/screenshots/`
- a beginner-facing article under `OriginOne-Wiki/article/`
- design package validation files under `OriginOne-Wiki/design-package/`
